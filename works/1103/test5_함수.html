함수연습

    0         1         2         3
    0123456789012345678901234567890
    ABCDEFGHIJKLMNOPQRSTUVWXYZABCopq

length : 객체(문자열)의 길이

slice() : 지정된 문자열을 추출해준다.(잘라낸다)

substring(a,b) : 지정된 문자열(a에서 b앞까지 추출해준다.)

substr() :

replace(a,b) : a를 b로치환

replaceAll(a,b) : a를 b로 모두 치환

toUpperCase() : 대문자로 치환

toLowerCase() : 소문자로 치환

concat(a,b) : a와 b를 결합한다.

trim() : 양쪽 공백을 삭제한다.

trimStart() : 앞쪽 공백을 삭제한다.

trimEnd() : 뒤쪽 공백을 삭제한다.

charAt() : 문자열의 문자를 하나만 가져와서, 문자로 바꿈.

charCodeAt(): .. 아스키 코드 값으로 출력

split() : 구분자를 기준으로 잘라준다. 잘라진 각각의 요소는 배열로 저장된다.

indexOf() : 특정문자의 위치값을 찾아준다.

lastIndexOf() : 특정문자의 위치값을 뒤에서부터 찾아준다.

search() : 특정문자의 위치값을 찾아준다(찾는 값이 없으면 -1 출력)

match() : 찾는 문자열이 있으면 '문자열'을 반환, 없으면 '공백'을 반환함.

includes() : 찾는 문자열을 포함하고 있으면 true, 없으면 flase를 반환한다.
