Display 속성

Display속성은 각각의 요소를 어떻게 보여줄지를 결정한다.

속성처리값 ? visibility(기본값:보이기) / hidden(감추기) / inherit(부모요소의 값을 상속받아 사용)

none : 보여주지 않음.
inline : 인라인상태(자기 자신의 고유한 영역만 확보) - 단어의 의미로 해석하면 될듯.. (크기조정 불가)
block : 박스영역을 차지하는 속성 - 문단의 의미로 해석하면 될듯.. (크기조정 가능)
inline-block : inline의 속성과 block의 속성을 합성
