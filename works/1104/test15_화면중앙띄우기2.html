화면 중앙에 새창 띄우기(부모창)

텍스트박스에 윈도우 가로크기와 세로크기를 입력받아, 버튼을 클릭하면 화면 중앙에 사용자가 지정한 사이즈의 새창(t15_child.html)을 띄우시오. 단, 최소 창의 크기는(300px * 200px)

윈도우 가로크기:
윈도우 세로크기: 

      : 새창에 사용되는 이름(name) ?
          _blank : 새창(기본값), _parent : 부모프레임에 열린다. _top : 로드된 프레임셋을 다시 대체한다. _self : 자신의 창에 띄운다.
      : 새창에 사용되는 옵션(option)?
          width : 창의 폭, height : 창의 높이, left : x축의 위치, top : y축의 위치
          fullscreen=(yes/no/auto) : 전체화면 모드 (이전에 됐었는데, 언제부턴가 지원을 안하는 브라우저가 많이 생김.)
          titlebar=(yes/no/auto) : 제목 표시줄의 표시 여부를 결정
          location=(yes/no/auto) : 주소 표시줄의 출력 여부를 결정
          menubar=(yes/no/auto) : 메뉴 표시줄의 표시 여부를 결정
          resizable=(yes/no/auto) : 창의 크기를 제어 여부 결정
          scrollbars=(yes/no/auto) : 스크롤바의 표시 여부 결정
          toolbar=(yes/no/auto) : 툴바(도구상자)의 표시 여부 결정
          현재 지원되지 않는 브라우저가 되게 많아지고 있는 추세라, 되는 명령이 있고, 되지 않는 명령이 있다.
